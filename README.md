# 📊 Power BI Sales Store Dashboard

## 📌 Project Overview
This project is an interactive Sales Store Dashboard created using Power BI.
It helps analyze sales performance, profit trends, and customer insights.

## 🎯 Objectives
* Monitor overall sales performance
* Identify top-performing products
* Analyze profit and loss
* Understand regional sales trends

## 🛠 Tools & Technologies
* Power BI
* Excel / CSV Dataset

## 📊 Dashboard Features

* Sales by Region
* Profit Analysis
* Category-wise Sales
* Time-based Trends
